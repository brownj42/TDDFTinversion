derivedtypes

This is the module that contains the definitions of the derived types and the subroutines that initialize and destroy the allocatable arrays therein.

Derived types

There are four derived types that are used in the program.

  • systemparameters: This contains all the information that defines the system being studied (mandatory)
  • ksvalues: This contains all the information about the Kohn-Sham system (mandatory)
  • sharedvalues: This contains the one-body potential, time-independant 1-body (mandatory) potential, the 2-body interaction potential. (mandatory)
  • fullvalues: This contains the full n-body wavefunction and the n-body (optional) potential
